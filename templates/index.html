About the Phishing Detection System

The Phishing Detection and Awareness System is a web-based cybersecurity tool designed to help users identify potentially malicious URLs and educate them about phishing threats.

Leveraging a machine learning model (XGBoost), the system analyzes URLs using features such as URL length, domain age, HTTPS presence, and suspicious keywords to classify them as phishing or legitimate.

  • Real-time detection with FastAPI backend
  • User-friendly frontend (HTML, CSS, JS)
  • Educational tips on phishing awareness
  • High accuracy using diverse dataset sources (PhishTank, OpenPhish, Alexa Top Sites, Kaggle)
